# Minimum Viable Development Stack

For any prototype and small applications, you msut use this stack:

- Go for backend
- SQLite for database using (modernc.org/sqlite)
- Go+Templ (https://github.com/a-h/templ)+HTMX (https://htmx.org/reference/)
