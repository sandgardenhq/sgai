# Web app workspaces

This page explains how to pin and unpin a root workspace when you are working in forked mode.

## Prerequisites

- Access to the web app.
- A root workspace that has forks.

## Pin or unpin a root workspace

1. Open the root workspace in the web app.
2. Find the **Pin** button.
3. Select **Pin** to pin the workspace.
4. Select **Unpin** to unpin the workspace.

The button reflects the current state:

- **Pin** shows when the workspace is not pinned.
- **Unpin** shows when the workspace is pinned.
