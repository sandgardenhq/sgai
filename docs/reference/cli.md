# CLI commands

This page describes the `sgai` command-line interface.

## Usage

```sh
sgai [--listen-addr addr]
```

Run `sgai` to start the web server.

## Global options

## `--listen-addr`

Set the HTTP server listen address.

Default: `127.0.0.1:8080`

## Commands

`sgai` does not expose additional subcommands in this reference set.
