# Environment variables

This page lists environment variables that `sgai` reads.

## `SGAI_MCP_URL`

Sets the MCP server URL for agent processes.

`sgai` sets this when launching agent processes.

## `SGAI_AGENT_IDENTITY`

Sets the agent identity for the MCP integration.

`sgai` sets this when launching agent processes.

## `SGAI_NTFY`

If `SGAI_NTFY` is set, `sgai` sends remote notifications by posting the message body to that URL.
