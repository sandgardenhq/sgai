Coordinator-first agent drive

Only chosen agents enter the room.

SGAI launches exactly one top-level worker: the coordinator. The GOAL file feeds it a safe payload of available OpenCode subagents. Anything outside the configured allowlist is stopped before prompt construction.

1

GOAL frontmatter is small: an agents list and one shared model.

2

The coordinator is implicit: it is launched by SGAI, not delegated as a worker.

3

STPA stays a skill workflow: stpa-analyst is never a routable GOAL agent.
